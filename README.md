# Flutter E-Commerce App

A Flutter-based e-commerce application with clean UI and core shopping features.

## Features
- User Login & Authentication
- Product Listing
- Add to Cart
- Cart Update Snackbar
- Responsive UI
- Android, iOS & Web Support

## Tech Stack
- Flutter
- Dart
- GetX (State Management)
- REST API
